# PICO MR: Video See-Through, LUTs, and Custom Toggles

![PICO XR](https://img.shields.io/badge/Platform-PICO%20XR-blue)
![Unity](https://img.shields.io/badge/Unity-6.0-black)

This repository contains the official Unity project and source code for the tutorial: **"Mastering PICO Video See-Through."** Learn how to implement Mixed Reality features, manage hardware toggles via C#, and apply cinematic LUT textures to your passthrough feed.

## 📺 Watch the Tutorial
[Click here to watch the full step-by-step guide on YouTube](https://youtu.be/4dVGKAq52QQ)

## 🚀 Key Features
* **Manual MR Toggle:** A optimized C# script to flip between VR and MR using controller input actions.
* **Cinematic LUTs:** Sample lookup textures and the correct import settings for PICO hardware.
* **Building Blocks Setup:** Implementation using the latest PICO Building Blocks for fast iteration.
* **Performance Fixes:** Ready-to-use project settings to avoid the common "Black Screen" (Mobile HDR) bug.
